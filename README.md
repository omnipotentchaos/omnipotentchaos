## 🚀 Featured Projects & Architecture

Here is a dive into the systems I've engineered, showcasing my ability to handle everything from complex backend data pipelines to sleek client-side interfaces.

### 🧠 AI & Machine Learning Systems
* **[Research Lens]** - An advanced Python & Next.js platform designed to parse and analyze academic literature. 
    * *Features:* Implements complex Python pipelines for embedding generation, clustering, knowledge graph creation, and temporal literature analysis using Vis-network and D3 concepts.
* **[LegalSummary AI]** - A Next.js application leveraging Firebase and Google Cloud. 
    * *Features:* Uses AI to rewrite complex legal jargon into plain language, extract key terms, detect unfair clauses, and utilize Google Cloud for Speech-to-Text and Data Loss Prevention (DLP).
* **[CredexAudit]** - A lightning-fast, client-side Next.js 16 (App Router) engine.
    * *Features:* Evaluates AI tool spend across major LLMs (Claude, GPT, Gemini). Integrates the Cerebras API (Llama 3.1 8B) for ultra-fast, 2200-tokens/s summaries, backed by Supabase and Vitest.
* **[PrimeTrade AI]** - An algorithmic trading analysis backend.
    * *Features:* Built with Python, featuring data loaders, analysis modules, and complex data visualizations wrapped in a Dockerized environment.

### ⚡ Real-Time & Communication Platforms
* **[InterviewPlatformJS]** - A comprehensive platform for technical interviews.
    * *Features:* Built with Next.js, Clerk (Auth), and Convex (Database). Integrates Stream API for real-time video meetings and includes a synchronized code editor with a dashboard for recording playbacks and comments.
* **[Vedaz Booking]** - A real-time booking and scheduling ecosystem.
    * *Features:* Utilizes a React (Vite) frontend with a Node/Express backend. Implements `Socket.io` for real-time updates between experts and users.

### 🏢 Full-Stack Marketplaces & FinTech
* **[GolfStake]** - A Next.js and Supabase application for golf tournament staking.
    * *Features:* Includes a custom prize calculator engine, charity donation routing, live score tracking, admin dashboards, and Stripe checkout webhook integration.
* **[Wanderlust]** - A comprehensive property booking platform.
    * *Features:* Built on the classic MERN-esque stack (Node, Express, MongoDB, EJS). Handles complex relational data like reviews, geographic map plotting, and user authentication schemas.
* **[Task Manager API]** - A highly structured Express & Node backend.
    * *Features:* Fully tested with Jest. Implements strict pagination, filtering schemas, data validation, and task assignment logic.
